22.	Написать скрипт который выполнит автоматически пункты 3, 4, 5, 6, 7, 8, 13

Создаем текстовый файл vim, называем myscript.sh
 в файле прописываем:

#!/bin/bash
mkdir Папка
cd Папка
mkdir 1 2 3
cd 1
touch 1.txt 2.txt 3.txt 1.json 2.json
mkdir A B C
ls -la
mv 2.txt 3.txt A

Затем в командной строке:

 ./Myscript.sh

22. Команда grep

 grep  ищет информацию в файле

cat file.txt | grep i слово    

будет искать слово в файле file.txt (i ставится чтобы искал в любом регистре), 
cat выводит строки с этим словом.
